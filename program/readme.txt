#in order to run the project you need to have the newest version of android studio

1. Install Android Studio with the newest version
2. Install Xampp
3. Import in the database from cocotravel.sql.The language in cocotravel.sql documentation is in Vietnamese but you
can ignore it and run the sql file
4. put all the folder inside htdocs folder into xamapp/htdocs
5. You have to use xampp and have it running while running the application, make sure you get your xampp 
ip address
6. watch video openapplication.mov to open the application in android studio
7. watch video replaceurl.mov to replace my ip address to your ip address
8. watch video runapplication.mov to run the app
